Runtime Complexity 

O(1) - Constant
e.g simple add numbers function

O(N) - Linear 
e.g looping numbers from 1 to n

O(LogN) - Logarithmic
e.g Finding an element in sorted array

O(N2) - Quadratic
e.g nested loops
